# CoffeeSales-Data-Analysis-Project
The Coffee Sales project focuses on analyzing vending machine transaction data using Data Analytics and Machine Learning techniques. The dataset contains detailed records of daily coffee sales from March 2024 to July 2024, with 1,133 transactions and 6 main columns: date, datetime, cash_type, card, money, and coffee_name. The objective of this project is to identify customer purchasing patterns, sales trends, and product performance to support better business decisions such as inventory planning and revenue optimization.

During data cleaning, missing values were identified in the “card” column (89 entries), which corresponded to cash payments. No duplicate records were found. The dataset includes 8 different coffee types, with Americano with Milk and Latte being the most popular products. Around 92% of transactions were made using cards, indicating strong digital payment usage.

Exploratory Data Analysis (EDA) revealed that Latte generated the highest revenue, while Espresso had the lowest. Monthly analysis showed increasing trends for Americano with Milk and Latte. Weekly sales indicated Tuesday as the highest-performing day. Hourly analysis identified two peak times: 10:00 AM and 7:00 PM, suggesting morning and evening demand spikes.

The project further demonstrates feature engineering, visualization using Matplotlib and Seaborn, and predictive modeling using Linear Regression. Overall, this project provides meaningful insights into customer behavior and supports data-driven business strategies.
